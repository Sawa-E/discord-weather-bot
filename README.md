# Discord 天気予報 Bot

毎日 0 時に東京の天気予報を Discord に自動投稿する Bot

## 機能

- 🌡️ 気温表示(朝・昼・夜、最低・最高)
- ☁️ 天気概況(雨のち晴れなど)
- 💧 降水確率
- 👕 おすすめの服装
- 🎒 持ち物アドバイス

## 技術スタック

- Python 3.13.9
- OpenWeatherMap API
- Discord Webhook
- Render (Cron Job)

## セットアップ

### 必要なもの

- OpenWeatherMap API キー
- Discord Webhook URL

### インストール

```bash
pip install -r requirements.txt
```
